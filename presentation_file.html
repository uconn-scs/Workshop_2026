Why msDiaLogue

To develop a flexible, customizable analysis pipeline for PMF users for use with proteomics experiments:

  • Primarily DIA-focussed, but expanded to DDA (Scaffold)
  • Adding peptide-level functionality (e.g., Phospo-proteome, Ubuquitome)
  • Including metabolomics

We want analysis choices to be as transparent as possible

See: https://uconn-scs.github.io/msDiaLogue/.

Overview

Aim to be able to conduct analysis from start to finish within an integrated platform:

  • Transformation
  • Normalization
  • Imputation
  • DA Analysis
  • Summarization & Visualization

Goals

  • To showcase some of the methodological features included in msDiaLogue.
  • Illustrate an analysis workflow on a ‘real’ dataset.
  • Promote the usage of this software for any proteomics (and metabolomics) analyses done in conjunction with PMF
  • Ask: any features you’d like to see?
